<h1>Countdown</h1>
<p>A simple Python program that takes a number from the user and prints a countdown from that number down to zero.</p>

<h2>Technologies Used</h2>
<ul>
  <li>Python 3</li>
  <li>Built-in <code>input()</code> function</li>
  <li>Built-in <code>print()</code> function</li>
  <li><code>while</code> loop</li>
  <li>Exception handling with <code>try</code> and <code>except</code></li>
</ul>

<h2>Features</h2>
<ul>
  <li>🔢 Accepts a number from the user</li>
  <li>✅ Validates input to ensure it is an integer</li>
  <li>🔁 Prints a countdown from the entered number to zero</li>
  <li>⚠️ Re-prompts the user when invalid input is entered</li>
  <li>📚 Beginner-friendly example of loops and input validation</li>
</ul>

<h2>Installation</h2>
<ol>
  <li>Make sure Python 3 is installed on your system.</li>
  <li>Download or copy the file <code>Countdown.py</code>.</li>
  <li>Open a terminal or command prompt in the project directory.</li>
  <li>Run the program using:</li>
</ol>

<pre><code>python Countdown.py</code></pre>

<p>The program will ask you to enter a number.</p>
<p>If the input is not a valid integer, the program will ask again until a valid number is provided.</p>
<p>After receiving a valid number, it prints each value one by one while decreasing it by 1 until it reaches zero.</p>

<h2>Output</h2>

<h3>Sample Input</h3>
<pre><code>number: 5</code></pre>

<h3>Sample Output</h3>
<pre><code>5
4
3
2
1
0</code></pre>

<h3>Invalid Input Example</h3>
<pre><code>number: abc
The entered age is not a number.
Re-enter the Age's life: 7</code></pre>

<h2>Notes</h2>
<ul>
  <li>⚠️ The input validation function is designed to ensure the entered value can be converted to an integer.</li>
  <li>💡 This project is useful for practicing loops, validation, and countdown logic in Python.</li>
</ul>
